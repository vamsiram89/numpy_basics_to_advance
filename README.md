# numpy_basics_to_advance

MOST IMPORTANT NUMPY COMMANDS (Data Science + Analytics)

1️⃣ Array Creation:-

| Command             | Use                                      |
| ------------------- | ---------------------------------------- |
| `np.array()`        --> Create array from list                   |
| `np.arange()`       --> Range of evenly spaced values            |
| `np.linspace()`     --> Evenly spaced values between start & end |
| `np.zeros()`        --> Array of zeros                           |
| `np.ones()`         --> Array of ones                            |
| `np.random.rand()`  --> Random uniform values                    |
| `np.random.randn()`-->| Random normal distribution               |
| `np.eye()`          --> Identity matrix                          |

2️⃣ Array Operations:-

| Command            | Use                         |
| ------------------ | --------------------------- |
| `np.add(a,b)`      --> Element-wise addition       |
| `np.subtract(a,b)` --> Element-wise subtraction    |
| `np.multiply(a,b)` -->|Element-wise multiplication |
| `np.divide(a,b)`   --> Element-wise division       |
| `np.power(a,2)`    --> Raise to power              |
| `np.sqrt(a)`       --> Square root                 |


3️⃣ Statistics:-

| Command       | Use                |
| ------------- | ------------------ |
| `np.mean()`   --> Average            |
| `np.median()` --> Median             |
| `np.std()`    --> Standard deviation |
| `np.var()`    --> Variance           |
| `np.sum()`    --> Sum                |
| `np.min()`    --> Minimum            |
| `np.max()`    --> Maximum            |

4️⃣ Shape & Reshape:-

| Command            | Use                    |
| ------------------ | ---------------------- |
| `arr.shape`        --> Get size (rows × cols) |
| `arr.reshape()`    --> Change shape           |
| `arr.flatten()`    --> Convert to 1D          |
| `arr.transpose()`  --> Swap rows/cols         |
| `np.expand_dims()` --> Add new dimension      |

5️⃣ Indexing & Filtering:-

| Command               | Use                   |
| --------------------- | --------------------- |
| `arr[rows, cols]`     --> Basic indexing        |
| `arr[1:5]`            --> Slicing               |
| `np.where(condition)` --> Conditional filtering |
| `np.unique()`         --> Unique values         |
| `np.argmax()`         --> Index of max          |
| `np.argmin()`         --> Index of min          |

6️⃣ Linear Algebra:-

| Command            | Use                       |
| ------------------ | ------------------------- |
| `np.dot(A, B)`     --> Dot/matrix multiplication |
| `np.linalg.inv(A)` --> Matrix inverse            |
| `np.linalg.det(A)` --> Determinant               |
| `np.linalg.eig(A)` --> Eigenvalues/vectors       |
